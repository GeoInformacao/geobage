# **Description**
geobage is an R package that allows users to easily access spatial data sets in the municipality of Bag√© / RS. The package includes a wide variety of geospatial data as simple resources. Were obtained from several sources for the acquisition of the original files in shapefile, later converted into GeoJSON, other geospatial data were generated by the sector of GeoInformation.

# **Installation**
```
# Use the development version with latest features
  utils::remove.packages('geobage')
  devtools::install_github("GeoInformacao/geobage")
  library(geobage)
```
Obs 1 - Before installing the package, clean the current session or start a new session.

Obs 2 - If you use Linux, you need to install a couple dependencies before installing the libraries sf, geobr and geojsonio. More info [here](https://github.com/r-spatial/sf#linux).

# **Basic Usage**
The syntax of all geobage functions operate from the same logic, making it intuitive to obtain data using a single line of code. Such that:
```
# Read shape municipality
mun <- read_shape_bg()

# Read all health facilitie of municipality Bag√©
mun <- read_health_facilities_bg()

# Read census tract (setor censit√°rio) of municipality Bag√©
mun <- read_census_tract_bg()
```
